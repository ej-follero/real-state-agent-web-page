# Marci Metzger Homes

A modern, responsive real estate agent website built with React, TypeScript, and Vite. This website showcases Marci Metzger's real estate services, properties, testimonials, and provides an interactive property search experience.

## Features

**Property Search** - Interactive property search with filtering capabilities
**Photo Gallery** - Beautiful image gallery showcasing properties
**Floating Chat** - Real-time chat widget for customer inquiries
**Interactive Maps** - Property location visualization
**Testimonials** - Client testimonials and reviews
**Contact Form** - Easy-to-use contact form for inquiries
**Responsive Design** - Fully responsive design for all devices
**Modern UI** - Built with Radix UI and Material UI components

## Tech Stack

**Framework**: React 18.3.1
**Language**: TypeScript 5.7.2
**Build Tool**: Vite 6.3.5
**Styling**: Tailwind CSS 4.1.12
**UI Components**: 
  - Radix UI (comprehensive component library)
  - Material UI 7.3.5
**State Management**: React Context API
**Form Handling**: React Hook Form 7.55.0
**Icons**: Lucide React, Material UI Icons
**Animations**: Motion 12.23.24
